# /public/assets/daki/

Directorio de assets visuales de la marca DAKI EdTech.

## Estructura esperada

```
daki/
├── logo.svg          — Logo principal DAKI EdTech (SVG, fondo transparente)
├── logo-dark.svg     — Variante oscura para fondos claros
├── logo-icon.svg     — Ícono solo (sin texto), para favicon/app icon
├── favicon.ico       — Favicon 32x32
├── og-image.png      — Open Graph image 1200x630 para redes sociales
└── daki-avatar.png   — Avatar de DAKI para el chat / onboarding
```

## Paleta oficial

| Token        | Hex       | Uso                          |
|--------------|-----------|------------------------------|
| neon-green   | `#00FF41` | Color primario, texto activo |
| terminal-bg  | `#0A0A0A` | Fondo principal              |
| surface      | `#0D0D0D` | Superficies/cards            |
| accent-amber | `#F59E0B` | Advertencias, ÉLITE badge    |

## Tipografía

- **Fira Code** (monoespaciada) — UI y código
- **Cascadia Code** / **Consolas** — Fallback
